# WordForge
Simple random password generator that takes parameters set to create simple passwords.

Currentley sett to use an inbuilt word list.

Parameters:
1. Word category.
2. Word Min and Max character length.
3. Symbol sets including custom.
4. number range to add to the end.

Future updates:
1. Import of custom wordlists.
2. Adding of capitals.
3. Creating random of letters + Numbers + Symbols in random order.
